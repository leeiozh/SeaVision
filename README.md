# SeaVision

## Обзор

**SeaVision** --- программный комплекс на Python для обработки
радиолокационных данных морской поверхности с целью восстановления
параметров волнения.

Система принимает: 
- сырые данные обратного рассеяния (*backscatter*) 
- навигационные параметры

и выполняет: 
- спектральный анализ 
- оценку характеристик волн

### Оцениваемые параметры

-   значительная высота волн *(Hs)*
-   пиковый период *(Tp)*
-   средний период *(Tm)*
-   направление волнения
-   направленный спектр
-   параметры ветра и течения

Поддерживаются режимы: 
- **реального времени (UDP)** 
- **офлайн-обработки (файлы)**

## Основные компоненты

### Manager (`runtime/manager.py`)

-   запуск потоков
-   управление очередями
-   перезапуск при ошибках
-   корректное завершение (EOF)

### Processor (`processing/processor.py`)

-   накопление окна (N_SHOTS)
-   секторизация
-   расчёт спектров
-   формирование результатов

### Averager (`processing/averaging.py`)

-   усреднение (MEAN)
-   spec_1d, spec_2d
-   параметры волн

### IO (`src/io`)

-   вход: UDP, файлы
-   выход: UDP, CSV

------------------------------------------------------------------------

## Алгоритм обработки

1.  Приём данных (`back.bck`, sog, cog, hdg)
2.  Секторизация → `cbck[sector, time, x, y]`
3.  FFT (пространство + время)
4.  ω--k спектр
5.  Доплер-коррекция
6.  Направленный спектр (`calc_spec2d`)
7.  Частотные параметры (m0, Tp, Tm)
8.  Декомпозиция (`calc_partitions`)
9.  Усреднение (MEAN)

------------------------------------------------------------------------

## Конфигурация

`config.ini`

### config.const

  Параметр   Описание
  ---------- ------------
  N_SHOTS    количество оборотов обработки
  NUM_AREA   количество секторов обработки
  N_FREQ     количество частот в спектре
  MEAN       параметр осреднения
  ADP, AAP   геометрия секторов обработки

### config.output

  Параметр    Описание
  ----------- ----------
  out_times   частота выходных параметров
